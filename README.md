# express-pug-vue-template
This is starter template using express, vue, element

## How to use
1. Clone this repo.
1. Execute `npm install`.
1. Execute `npm start`.
1. Execute `npm run sync` to use browser-sync
1. Enjoy.

- https://vuejs.org/
